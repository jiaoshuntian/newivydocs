## IvorySQL Cloud Platform

IvorySQL Cloud is a self-hosted web management platform that sits on top of the IvorySQL Operator. It gives you a graphical interface for provisioning, monitoring, and backing up IvorySQL databases — without writing Kubernetes manifests.

<Callout type="tip">
IvorySQL Cloud is self-hosted. You deploy it on your own Kubernetes cluster. Install the [IvorySQL Operator](/docs/v5/cloud-service/operator) first, then proceed with the platform installation below.
</Callout>
