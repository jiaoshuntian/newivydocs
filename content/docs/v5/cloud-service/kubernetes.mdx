# Kubernetes

IvorySQL ships Kubernetes manifests for both single-node and high-availability deployments. The HA option uses Patroni for automatic leader election and failover.

## Prerequisites

- A running Kubernetes cluster
- `kubectl` configured to reach the cluster
- Helm installed (HA cluster only)

---

## Single-node deployment

<Steps>

### Create the namespace

```bash
kubectl create ns ivorysql
```

### Clone the manifest templates

```bash
git clone https://github.com/IvorySQL/docker_library.git
cd docker_library/k8s-cluster/single
```

### Edit the StatefulSet

Open `statefulset.yaml` and update the PVC storage configuration and database password to match your environment:

```bash
vim statefulset.yaml
```

### Apply the manifest

```bash
kubectl apply -f statefulset.yaml
# service/ivorysql-svc created
# statefulset.apps/ivorysql created
```

### Wait for the pod to be ready

```bash
kubectl get all -n ivorysql
# NAME             READY   STATUS    RESTARTS   AGE
# pod/ivorysql-0   1/1     Running   0          2m39s
#
# NAME                   TYPE       CLUSTER-IP       PORT(S)
# service/ivorysql-svc   NodePort   10.108.178.236   5432:32106/TCP,1521:31887/TCP
```

### Connect

<Tabs items={['PostgreSQL port (pg mode)', 'Oracle-compatible port (oracle mode)']}>
<Tab>
```bash
psql -U ivorysql -p 32106 -h 127.0.0.1 -d ivorysql
```

Verify compatibility mode:

```sql
SHOW ivorysql.compatible_mode;
--  ivorysql.compatible_mode
-- --------------------------
--  pg
```
</Tab>
<Tab>
```bash
psql -U ivorysql -p 31887 -h 127.0.0.1 -d ivorysql
```

Verify compatibility mode:

```sql
SHOW ivorysql.compatible_mode;
--  ivorysql.compatible_mode
-- --------------------------
--  oracle
```
</Tab>
</Tabs>

### Uninstall

```bash
kubectl delete -f statefulset.yaml
```

</Steps>

## High-availability cluster (Helm)

The HA deployment uses [Patroni](https://patroni.readthedocs.io/) for automatic leader election. It requires [Helm](https://helm.sh/docs/intro/install/).

<Steps>

### Create the namespace

```bash
kubectl create ns ivorysql
```

### Clone the Helm chart templates

```bash
git clone https://github.com/IvorySQL/docker_library.git
cd docker_library/k8s-cluster/ha-cluster/helm_charts
```

### Configure the cluster

Edit `values.yaml` to adjust PVC settings, cluster size, and other options. For the database password, see `templates/secret.yaml`.

```bash
vim values.yaml
```

### Deploy with Helm

```bash
helm install ivorysql-ha-cluster -n ivorysql .
# NAME: ivorysql-ha-cluster
# STATUS: deployed
```

### Wait for all pods to be ready

```bash
kubectl get all -n ivorysql
```

Wait until all three Patroni pods show `1/1 Running`:

```
NAME                         READY   STATUS    RESTARTS   AGE
pod/ivorysql-patroni-hac-0   1/1     Running   0          88s
pod/ivorysql-patroni-hac-1   1/1     Running   0          64s
pod/ivorysql-patroni-hac-2   1/1     Running   0          41s
```

### Connect to the primary

<Tabs items={['PostgreSQL port (pg mode)', 'Oracle-compatible port (oracle mode)']}>
<Tab>
```bash
psql -U ivorysql -p 32391 -h 127.0.0.1 -d ivorysql
```

Confirm this is the primary and check mode:

```sql
SHOW ivorysql.compatible_mode;
--  ivorysql.compatible_mode
-- --------------------------
--  pg

SELECT pg_is_in_recovery();
--  pg_is_in_recovery
-- -------------------
--  f
```
</Tab>
<Tab>
```bash
psql -U ivorysql -p 32477 -h 127.0.0.1 -d ivorysql
```

Confirm this is the primary and check mode:

```sql
SHOW ivorysql.compatible_mode;
--  ivorysql.compatible_mode
-- --------------------------
--  oracle

SELECT pg_is_in_recovery();
--  pg_is_in_recovery
-- -------------------
--  f
```
</Tab>
</Tabs>

### Connect to a standby

<Tabs items={['PostgreSQL port (pg mode)', 'Oracle-compatible port (oracle mode)']}>
<Tab>

<Tab>

--  oracle

SELECT pg_is_in_recovery();
--  t

</Tab>
</Tabs>

### Uninstall

```bash
helm uninstall ivorysql-ha-cluster -n ivorysql
```

Remove the PVCs:

```bash
for i in 0 1 2; do
  kubectl delete pvc ivyhac-config-ivorysql-patroni-hac-$i -n ivorysql
  kubectl delete pvc pgdata-ivorysql-patroni-hac-$i -n ivorysql
done
```

</Steps>
