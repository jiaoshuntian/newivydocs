IvorySQL 是一个基于 PostgreSQL 构建的开源、Oracle 兼容数据库，由 Highgo Software 维护。它让你可以在不离开 PostgreSQL 生态的前提下，运行 Oracle SQL 语法、PL/SQL 存储过程和 Oracle 内置函数。
