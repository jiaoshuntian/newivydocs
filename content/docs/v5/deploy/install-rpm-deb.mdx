# Install with RPM / DEB Packages

This guide shows you how to install IvorySQL using pre-built RPM or DEB packages downloaded directly from GitHub releases.

## RPM Installation (CentOS / RHEL)

<Steps>

### Step 1: Install dependencies

```bash
sudo dnf install -y lz4 libicu libxslt python3
```

### Step 2: Download the RPM

```bash
sudo wget https://github.com/IvorySQL/IvorySQL/releases/download/IvorySQL_5.4/IvorySQL-5.4-9d890e9-20251120.x86_64.rpm
```

### Step 3: Install

```bash
sudo yum --disablerepo=* localinstall *.rpm
```

IvorySQL will be installed to `/usr/ivory-5/`.

### Step 4: Initialize and start

```bash
sudo chown -R ivorysql:ivorysql /usr/ivory-5/
mkdir /usr/ivory-5/data
initdb -D /usr/ivory-5/data
pg_ctl -D /usr/ivory-5/data -l ivory.log start
```

</Steps>

## DEB Installation (Ubuntu / Debian)

<Steps>

### Step 1: Install dependencies

```bash
sudo apt -y install pkg-config libreadline-dev libicu-dev libldap2-dev uuid-dev tcl-dev libperl-dev python3-dev bison flex openssl libssl-dev libpam-dev libxml2-dev libxslt-dev libossp-uuid-dev libselinux-dev gettext
```

### Step 2: Download the DEB

```bash
sudo wget https://github.com/IvorySQL/IvorySQL/releases/download/IvorySQL_5.4/IvorySQL-5.4-a50789d-20250304.amd64.deb
```

### Step 3: Install

```bash
sudo dpkg -i IvorySQL-5.4-a50789d-20250304.amd64.deb
```

### Step 4: Initialize and start

```bash
mkdir /usr/ivory-5/data
initdb -D /usr/ivory-5/data
pg_ctl -D /usr/ivory-5/data -l ivory.log start
```

</Steps>

## Uninstallation

<Tabs>
<Tab title="RPM" value="rpm">

```bash
sudo yum remove --disablerepo=* ivorysql5\*
sudo rm -rf /usr/ivory-5/
```

</Tab>
<Tab title="DEB" value="deb">

```bash
sudo dpkg -P IvorySQL-5.4-a50789d-20250304.amd64.deb
sudo rm -rf /usr/ivory-5/
```

</Tab>
</Tabs>
