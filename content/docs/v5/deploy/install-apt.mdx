# Install with APT / DEB

This guide shows you how to install IvorySQL on Ubuntu and Debian-based distributions using a pre-built `.deb` package from GitHub Releases.

<Callout type="info">
IvorySQL does not currently publish an APT repository. Installation on Debian/Ubuntu uses a `.deb` package installed directly with `dpkg`.
</Callout>

## Prerequisites

Create a user with root privileges. All installation steps should be performed as this user (we'll use `ivorysql` as the example username).

## Installation

<Steps>

### Step 1: Install dependencies

```bash
sudo apt -y install pkg-config libreadline-dev libicu-dev libldap2-dev uuid-dev \
  tcl-dev libperl-dev python3-dev bison flex openssl libssl-dev libpam-dev \
  libxml2-dev libxslt-dev libossp-uuid-dev libselinux-dev gettext
```

### Step 2: Download the DEB package

```bash
sudo wget https://github.com/IvorySQL/IvorySQL/releases/download/IvorySQL_5.4/IvorySQL-5.4-a50789d-20250304.amd64.deb
```

Check the [IvorySQL releases page](https://github.com/IvorySQL/IvorySQL/releases) for the latest package filename.

### Step 3: Install the package

```bash
sudo dpkg -i IvorySQL-5.4-a50789d-20250304.amd64.deb
```

IvorySQL will be installed to `/usr/ivory-5/`.

### Step 4: Grant permissions

```bash
sudo chown -R ivorysql:ivorysql /usr/ivory-5/
```

### Step 5: Set environment variables

Add to `~/.bash_profile`:

```bash
PATH=/usr/ivory-5/bin:$PATH
export PATH
PGDATA=/usr/ivory-5/data
export PGDATA
```

Source the file:

```bash
source ~/.bash_profile
```

### Step 6: Initialize the database

```bash
mkdir /usr/ivory-5/data
initdb -D /usr/ivory-5/data
```

### Step 7: Start the service

```bash
pg_ctl -D /usr/ivory-5/data -l ivory.log start
```

### Step 8: Verify the service is running

```bash
ps -ef | grep postgres
```

You should see `postgres` processes including `checkpointer`, `background writer`, `walwriter`, and `autovacuum launcher`.

### Step 9: Connect to IvorySQL

```bash
psql -d ivorysql
```

You will see the `ivorysql=#` prompt. Type `\q` to exit.

</Steps>

## Uninstallation

<Callout type="warn">
Ensure the service is stopped and your data is backed up before uninstalling.
</Callout>

Stop the service first:

```bash
pg_ctl stop
```

Then remove the package and data:

```bash
sudo dpkg -P IvorySQL-5.4-a50789d-20250304.amd64.deb
sudo rm -rf /usr/ivory-5/
```
