# Install with APT

This guide shows you how to install IvorySQL on Ubuntu and Debian-based distributions using APT.

## Prerequisites

Create a user with root privileges for all installation steps.

## Installation

<Steps>

### Step 1: Download the DEB package

```bash
sudo wget https://github.com/IvorySQL/IvorySQL/releases/download/IvorySQL_5.4/IvorySQL-5.4-a50789d-20250304.amd64.deb
```

### Step 2: Install the package

```bash
sudo dpkg -i IvorySQL-5.4-a50789d-20250304.amd64.deb
```

IvorySQL will be installed to `/usr/ivory-5/`.

### Step 3: Set environment variables

Add to `~/.bash_profile`:

```bash
PATH=/usr/ivory-5/bin:$PATH
export PATH
PGDATA=/usr/ivory-5/data
export PGDATA
```

Source the file:

```bash
source ~/.bash_profile
```

### Step 4: Initialize the database

```bash
mkdir /usr/ivory-5/data
initdb -D /usr/ivory-5/data
```

### Step 5: Start the service

```bash
pg_ctl -D /usr/ivory-5/data -l ivory.log start
```

### Step 6: Verify

```bash
ps -ef | grep postgres
```

</Steps>

## Uninstallation

<Callout type="warn">
Ensure the service is stopped and data is backed up first.
</Callout>

```bash
sudo dpkg -P IvorySQL-5.4-a50789d-20250304.amd64.deb
sudo rm -rf /usr/ivory-5/
```
