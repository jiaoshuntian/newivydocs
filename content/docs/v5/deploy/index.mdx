# Deploy

Install IvorySQL on Linux using the package manager of your distribution, pre-built binaries, or by compiling from source. Once installed, optionally configure streaming replication for high availability.

<Callout type="tip">
**Deploying with containers or Kubernetes?** See the [Cloud Service](/docs/v5/cloud-service) section — it covers Docker, Podman, Docker Swarm, Kubernetes, and the IvorySQL Operator.
</Callout>
