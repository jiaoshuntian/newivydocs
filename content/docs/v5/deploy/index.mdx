# Deploy

Install IvorySQL on Linux, build from source, or deploy with Docker — this section covers every method.

<Callout type="tip">
**Not sure which method to use?**
- **CentOS / RHEL** → [YUM / DNF](/docs/v5/deploy/install-yum) — easiest, managed updates
- **Ubuntu / Debian** → [APT / DPKG](/docs/v5/deploy/install-apt)
- **Containers** → start with the [Quick Start Docker path](/docs/v5/getting-started/quick-start)
- **Air-gapped or custom builds** → [RPM / DEB packages](/docs/v5/deploy/install-rpm-deb) or [Build from Source](/docs/v5/deploy/install-source)
- **Production HA** → [Cluster Setup](/docs/v5/deploy/cluster) after initial install
</Callout>
