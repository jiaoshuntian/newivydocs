# Ecosystem

IvorySQL is built on PostgreSQL, which means it inherits one of the most powerful extension ecosystems in the database world. Beyond full compatibility with PostgreSQL extensions, the IvorySQL team has tested and validated a curated set of plugins for geospatial processing, vector retrieval, full-text search, scheduled tasks, HTTP integration, code quality auditing, and system monitoring.

Each extension listed below has been verified to run stably in the IvorySQL environment. Select a plugin based on your business needs to further enhance the capabilities and flexibility of your database.
