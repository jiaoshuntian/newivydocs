# system_stats

system_stats is a PostgreSQL extension that provides functions to access system-level statistics, useful for monitoring and diagnostics. It exposes operating system metrics directly through SQL functions.

## Installation

<Callout type="info">
These steps assume Ubuntu 24.04 (x86_64) with IvorySQL v5+ installed at `/usr/local/ivorysql/ivorysql-5`.
</Callout>

<Steps>

### Step 1: Download and build

Download the [v3.2 source package](https://github.com/EnterpriseDB/system_stats/releases/tag/v3.2):

```bash
unzip v3.2.zip
cd system_stats-3.2

make PG_CONFIG=/usr/local/ivorysql/ivorysql-5/bin/pg_config
make PG_CONFIG=/usr/local/ivorysql/ivorysql-5/bin/pg_config install
```

</Steps>

## Enable and Verify

```sql
CREATE EXTENSION system_stats;

SELECT * FROM pg_available_extensions WHERE name = 'system_stats';
```

Expected output:

```
     name     | default_version | installed_version |                    comment
--------------+-----------------+-------------------+-----------------------------------------------
 system_stats | 3.0             | 3.0               | EnterpriseDB system statistics for PostgreSQL
(1 row)
```

## Usage

The extension provides the following functions:

### OS Information

```sql
SELECT pg_sys_os_info();
```

```
                              pg_sys_os_info
--------------------------------------------------------------------------
 ("""Ubuntu 24.04.1 LTS""                                                +
 ","Linux 6.14.0-29-generic",Ubuntu,"(none)",5568,292,532,x86_64,,864122)
(1 row)
```

### CPU Information

```sql
SELECT pg_sys_cpu_info();
```

```
                                                           pg_sys_cpu_info
-------------------------------------------------------------------------------------------------------------------------------------
 (GenuineIntel,"GenuineIntel model 154 family 6","12th Gen Intel(R) Core(TM) i7-1260P",,0,2,1,x86_64,2496000000,,,,48,32,1280,18432)
(1 row)
```

### CPU Usage

```sql
SELECT pg_sys_cpu_usage_info();
```

```
 pg_sys_cpu_usage_info
-----------------------
 (0,0,0,100,0,0,0,,,,)
(1 row)
```

### Memory

```sql
SELECT pg_sys_memory_info();
```

```
                                pg_sys_memory_info
-----------------------------------------------------------------------------------
 (4055482368,3911159808,144322560,4055887872,233725952,3822161920,2638602240,,,,,)
(1 row)
```

### Disk I/O

```sql
SELECT pg_sys_io_analysis_info();
```

### Disk Space

```sql
SELECT pg_sys_disk_info();
```

### Load Average

```sql
SELECT pg_sys_load_avg_info();
```

### Process Info

```sql
SELECT pg_sys_process_info();
```

### Network

```sql
SELECT pg_sys_network_info();
```

### Per-Process CPU/Memory

```sql
SELECT pg_sys_cpu_memory_by_process();
```
