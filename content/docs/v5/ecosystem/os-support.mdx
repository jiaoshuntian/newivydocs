# Operating System Support

IvorySQL has been tested and certified on the following operating systems:

| OS | Description |
|---|---|
| Kylin Advanced Server OS V11 | A next-generation server operating system designed for enterprise-level critical business by Kylin Software. Features high reliability, availability, security, and performance with MPTCP high-speed networking, hot patch management, and intelligent fault diagnosis. |
| openKylin 2.0 SP1 | An open-source project incubated by the OpenAtom Foundation, built with the vision of providing an open-source operating system deeply integrated with AI technology. |
| OpenAnolis OS 23 | An enterprise-level operating system built on Linux Kernel 6.6 LTS, with in-depth optimization of the ANCK kernel. Fully supports domestic chips including Haiguang, Phytium, Loongson (LoongArch), and Zhaoxin, as well as x86_64/ARM64 architectures. |
| Deepin 20 | A Linux distribution providing an elegant, user-friendly, and secure operating system. Based on Debian 10.5 with a dual-kernel mechanism (5.4 / 5.7) for enhanced stability and compatibility. |

<Callout type="info">
If your target OS is not listed, IvorySQL can still be built from source on most Linux distributions. See the [Build from Source](/docs/v5/deploy/install-source) guide for instructions.
</Callout>
