title: IvorySQL Documentation

---

IvorySQL is an open-source, Oracle-compatible database built on PostgreSQL, maintained by Highgo Software. It lets you run Oracle SQL syntax, PL/SQL stored procedures, and Oracle built-in functions without leaving the PostgreSQL ecosystem.
