# Quick Start

Pick your platform and have a working IvorySQL instance in under 5 minutes.

<Tabs items={['CentOS / RHEL', 'Ubuntu / Debian', 'Docker']}>

<Tab value="CentOS / RHEL">

<Steps>

### Add the IvorySQL repository

Create `/etc/yum.repos.d/ivorysql.repo`:

```ini
[ivorysql5]
name=IvorySQL Server 5 $releasever - $basearch
baseurl=https://yum.highgo.com/dists/ivorysql-rpms/5/redhat/rhel-$releasever-$basearch
enabled=1
gpgcheck=0
```

### Install IvorySQL

```bash
sudo dnf install -y ivorysql5-server
```

### Set environment variables

Add to `~/.bash_profile`:

```bash
PATH=/usr/ivory-5/bin:$PATH
export PATH
PGDATA=/usr/ivory-5/data
export PGDATA
```

```bash
source ~/.bash_profile
```

### Initialize and start

```bash
initdb -D /usr/ivory-5/data
pg_ctl -D /usr/ivory-5/data -l ivory.log start
```

### Connect

```bash
psql -d ivorysql
```

You should see the `ivorysql=#` prompt.

</Steps>

</Tab>

<Tab value="Ubuntu / Debian">

<Steps>

### Add the IvorySQL repository

```bash
sudo apt-get install -y curl gnupg
curl -fsSL https://apt.highgo.com/ivorysql.gpg | sudo gpg --dearmor -o /etc/apt/trusted.gpg.d/ivorysql.gpg
echo "deb https://apt.highgo.com/dists/ivorysql-debs/5/ubuntu focal main" | sudo tee /etc/apt/sources.list.d/ivorysql.list
sudo apt-get update
```

### Install IvorySQL

```bash
sudo apt-get install -y ivorysql5-server
```

### Initialize and start

```bash
initdb -D /usr/ivory-5/data
pg_ctl -D /usr/ivory-5/data -l ivory.log start
```

### Connect

```bash
psql -d ivorysql
```

</Steps>

</Tab>

<Tab value="Docker">

<Steps>

### Pull the image

```bash
docker pull ivorysql/ivorysql:latest
```

<Callout type="tip">
Users in China can also pull from the IvorySQL community Harbor registry for faster downloads: `docker pull registry.highgo.com/ivorysql/ivorysql:latest`
</Callout>

### Run the container

```bash
docker run --name ivorysql \
  -p 5434:5432 \
  -e IVORYSQL_PASSWORD=your_password \
  -d ivorysql/ivorysql:latest
```

### Verify it started

```bash
docker ps | grep ivorysql
```

### Connect

```bash
psql -d ivorysql -U ivorysql -h 127.0.0.1 -p 5434
```

</Steps>

</Tab>

</Tabs>

## Try Oracle compatibility

Once connected, run a quick Oracle-syntax smoke test:

```sql
-- Oracle-style dual table
SELECT SYSDATE FROM dual;

-- PL/iSQL anonymous block
BEGIN
  RAISE NOTICE 'IvorySQL Oracle mode is working!';
END;
/
```

<Callout type="info">
IvorySQL defaults to Oracle-compatible mode. Connect on port 1521 (or set `ivorysql.compatible_mode = oracle`) to ensure Oracle syntax is active.
</Callout>
