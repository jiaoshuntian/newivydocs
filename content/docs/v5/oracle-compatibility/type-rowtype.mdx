# %TYPE & %ROWTYPE

IvorySQL provides Oracle-compatible `%TYPE` and `%ROWTYPE` declarations, allowing variables to inherit types from table columns or entire row structures.

## %TYPE

### Referencing Changes

When a referenced column changes, variables declared with `%TYPE` are automatically invalidated and re-resolved:

```sql
CREATE TABLE t1(id int, name varchar(20));

CREATE OR REPLACE FUNCTION fun1(v t1.id%TYPE) RETURN varchar AS
BEGIN
  RETURN v;
END;
/

-- Function is valid
SELECT prostatus FROM pg_proc WHERE proname LIKE 'fun1';
--  prostatus
-- -----------
--  v

SELECT fun1(1) FROM dual;
--  fun1
-- ------
--  1

-- Alter the column type
ALTER TABLE t1 ALTER COLUMN id TYPE varchar(20);

-- Function becomes invalid (needs recompilation)
SELECT prostatus FROM pg_proc WHERE proname LIKE 'fun1';
--  prostatus
-- -----------
--  n

-- But it still executes successfully with the new type
SELECT fun1('a') FROM dual;
--  fun1
-- ------
--  a

-- Recompile
ALTER FUNCTION fun1 COMPILE;
SELECT prostatus FROM pg_proc WHERE proname LIKE 'fun1';
--  prostatus
-- -----------
--  v
```

### Inheriting Constraints

Variables declared with `%TYPE` inherit constraints from the referenced variable:

```sql
-- This fails because surname inherits NOT NULL but has no default
DECLARE
  name     VARCHAR(25) NOT NULL := 'Niu';
  surname  name%TYPE;
BEGIN
  RAISE NOTICE 'name=%', name;
  RAISE NOTICE 'surname=%', surname;
END;
/
-- ERROR: variable "surname" must have a default value, since it's declared NOT NULL
```

## %ROWTYPE

### Using as Parameter or Return Type

```sql
CREATE TABLE employees(
  first_name varchar(20) NOT NULL,
  last_name varchar(20) NOT NULL,
  phone_number varchar(50)
);

INSERT INTO employees VALUES ('Steven', 'Niu', '1-650-555-1234');

CREATE OR REPLACE PROCEDURE p0(v employees%ROWTYPE) AS
BEGIN
  RAISE NOTICE 'v.first_name = %, v.last_name = %, v.phone_number = %',
    v.first_name, v.last_name, v.phone_number;
END;
/

DECLARE
  a employees%ROWTYPE;
BEGIN
  SELECT * INTO a FROM employees;
  RAISE NOTICE 'a=%', a;
  CALL p0(a);
END;
/
-- NOTICE:  a=(Steven,Niu,1-650-555-1234)
-- NOTICE:  v.first_name = Steven, v.last_name = Niu, v.phone_number = 1-650-555-1234
```

### INSERT with %ROWTYPE

Insert a `%ROWTYPE` variable directly into a table:

```sql
CREATE TABLE t1(id int, name varchar(20));

DECLARE
  v1 t1%ROWTYPE;
BEGIN
  FOR i IN 1 .. 5 LOOP
    v1.id := i;
    v1.name := 'a' || i;
    INSERT INTO t1 VALUES v1;
  END LOOP;
END;
/

SELECT * FROM t1;
--  id | name
-- ----+------
--   1 | a1
--   2 | a2
--   3 | a3
--   4 | a4
--   5 | a5
```

### UPDATE with %ROWTYPE

Update an entire row using `SET ROW`:

```sql
CREATE TABLE t1(id int, name varchar(20));
DELETE FROM t1;

DECLARE
  v1 t1%ROWTYPE;
  v2 t1%ROWTYPE;
BEGIN
  v1.id := 11;
  v1.name := 'abc';
  INSERT INTO t1 VALUES v1;
  v2.id := 22;
  v2.name := 'new';
  UPDATE t1 SET ROW = v2;
END;
/

SELECT * FROM t1;
--  id | name
-- ----+------
--  22 | new
```
