# Ports & IP

IvorySQL provides separate port and IP handling for Oracle-compatible connections, keeping them distinct from standard PostgreSQL connections.

## How It Works

- **`ivoryhost`** — A connection parameter similar to `host`, used specifically for Oracle-compatible mode connections.
- **`ivoryport`** — A dedicated port for Oracle-compatible connections. This is separate from the standard PostgreSQL port and can be specified at configure, initialize, and runtime stages.

## Configuration

### At configure time

```bash
./configure --with-oraport=5555
```

### Initialize and start

```bash
./initdb -D ../data
./pg_ctl -D ../data start
```

### Specify ports at runtime

```bash
./pg_ctl -o "-p 5433 -o 1522" -D ../data
```

The first `-p` specifies the standard PostgreSQL port, and the second `-o` specifies the Oracle-compatible mode port.

## Default Ports

- **PostgreSQL port**: 5432
- **Oracle-compatible port**: 1521 (default, configurable)

You can check the current port settings:

```sql
SHOW ivorysql.port;
```
