# Empty String to NULL

In Oracle, an empty string (`''`) is treated as `NULL`. PostgreSQL treats them as distinct values. IvorySQL replicates the Oracle behavior in Oracle compatibility mode, so that applications expecting Oracle semantics work correctly without modification.

## How It Works

The feature is controlled by the GUC parameter `ivorysql.enable_emptystring_to_null`:

| Setting | Behavior |
|---|---|
| `on` (default in Oracle mode) | Empty strings are stored as `NULL` |
| `off` | Empty strings are stored as empty strings (PostgreSQL behavior) |

When enabled, any inserted empty string is automatically converted to `NULL` for storage. Converted `NULL` values can be queried using the `IS NULL` condition.

<Callout type="warn">
  This behavior differs from standard PostgreSQL, where `''` and `NULL` are distinct. Be aware of this when mixing Oracle-mode and PostgreSQL-mode access to the same data.
</Callout>

## Checking and Setting the Parameter

```sql
-- Check the current setting
SHOW ivorysql.enable_emptystring_to_null;
--  ivorysql.enable_emptystring_to_NULL
-- -------------------------------------
--  on

-- Disable the conversion (use PostgreSQL behavior)
SET ivorysql.enable_emptystring_to_null = off;

-- Re-enable Oracle behavior
SET ivorysql.enable_emptystring_to_null = on;
```

## Example

```sql
-- Create test table
CREATE TABLE abc (id int);

-- Verify parameter is enabled
SHOW ivorysql.enable_emptystring_to_null;
--  ivorysql.enable_emptystring_to_NULL
-- -------------------------------------
--  on

-- Insert an empty string
INSERT INTO abc VALUES('');
-- INSERT 0 1

-- Query the data — it shows as NULL (the empty string was converted)
SELECT * FROM abc;
--  id
-- ----
--
-- (1 row)

-- Confirm the value is NULL, not an empty string
SELECT * FROM abc WHERE id IS NULL;
--  id
-- ----
--
-- (1 row)

-- An empty-string comparison returns nothing (the value is NULL, not '')
SELECT * FROM abc WHERE id = '';
-- (0 rows)
```

## Oracle vs PostgreSQL Comparison

```sql
-- Oracle behavior (ivorysql.enable_emptystring_to_null = on)
INSERT INTO abc VALUES('');
SELECT id IS NULL FROM abc;  -- true

-- PostgreSQL behavior (ivorysql.enable_emptystring_to_null = off)
INSERT INTO abc VALUES('');
SELECT id IS NULL FROM abc;  -- false ('' is not NULL)
SELECT id = '' FROM abc;     -- true
```
