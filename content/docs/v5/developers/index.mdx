# Developers

This section covers IvorySQL's internal architecture, the implementation of built-in Oracle-compatible functions, and the design of the Oracle compatibility layer. It is intended for developers who want to understand how IvorySQL works under the hood, or for contributors looking to extend or modify the codebase.

IvorySQL builds on PostgreSQL by introducing a dual-parser framework, a custom procedural language (PL/iSQL), and a comprehensive set of Oracle-compatible features — all implemented as extensions to the PostgreSQL engine.
