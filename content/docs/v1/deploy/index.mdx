---
title: "Deploy IvorySQL"
description: "Installation options for IvorySQL 1.x"
---

IvorySQL 1.x can be deployed on Linux systems using two primary methods: building from source code or installing a pre-built binary package (RPM for CentOS/RHEL, DEB for Ubuntu). Both methods result in a fully functional IvorySQL instance with Oracle compatibility features enabled.

Choose the method that best fits your environment. Source builds give you full control over compile-time options; package installs are faster and recommended for standard deployments.
