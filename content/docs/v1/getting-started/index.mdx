**IvorySQL** is the only Oracle-compatible open-source PostgreSQL. It is proposed and maintained by Highgo Software, with an active open-source community. IvorySQL adds Oracle compatibility features on top of PostgreSQL without sacrificing PostgreSQL compatibility — you get both worlds in one database.

IvorySQL is open source and free to use. For questions or feedback, contact [support@ivorysql.org](mailto:support@ivorysql.org).

<Callout type="info">
This is the documentation for **IvorySQL v1.x**, which is based on **PostgreSQL 14**. If you are looking for a newer version, use the version switcher in the navigation bar.
</Callout>

## Getting the Code

Download the source code from [GitHub](https://github.com/IvorySQL/IvorySQL) or visit the [IvorySQL Release Page](https://www.ivorysql.org/releases-page) for pre-built packages.

## About the Project

IvorySQL is an open-source project proposed by Highgo Software to add Oracle compatibility features into the popular PostgreSQL database. It is open source, free to use, and committed to remaining 100% compatible with the latest PostgreSQL while offering seamless Oracle migration capabilities.
