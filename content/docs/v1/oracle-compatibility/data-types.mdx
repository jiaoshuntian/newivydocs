## Function reference

### sysdate

Returns the current system date.

```sql
SELECT sysdate() FROM dual;
--   sysdate
-- ------------
--  2023-07-06

SELECT sysdate()-1 FROM dual;
--   ?column?
-- ------------
--  2023-07-05
```

### systimestamp

Returns the current system date and time including microseconds and time zone.

```sql
SELECT systimestamp() FROM dual;
--   systimestamp
-- -----------------------------------
--  2023-07-06 10:18:31.674322 +08:00
```

### add_months

Adds `n` months to a date and returns the result.

```sql
SELECT add_months(sysdate(), 1) FROM dual;   -- next month same day
SELECT add_months(sysdate(), -1) FROM dual;  -- previous month same day
```

### last_day

Returns the last day of the month for the given date.

```sql
SELECT last_day(sysdate()) FROM dual;
-- 2023-07-31

SELECT last_day(to_date('2019-09-01')) FROM dual;
-- 2019-09-30
```

### next_day

Returns the next occurrence of a given weekday after the specified date. The second parameter can be an integer (1=Sunday) or a weekday name string.

```sql
SELECT next_day(sysdate(), 1) FROM dual;          -- next Sunday
SELECT next_day(sysdate(), 'FRIDAY') FROM dual;   -- next Friday
```

### months_between

Returns the number of months between two dates. Returns a positive number if `date1 > date2`, negative if `date1 < date2`. Returns an integer if both dates fall on the same day of their respective months; otherwise returns a decimal based on a 31-day month.

```sql
SELECT months_between(to_date('2023-07-06'), to_date('2023-08-06')) FROM dual;
-- -1

SELECT months_between(to_date('2023-07-06'), to_date('2023-08-05')) FROM dual;
-- -0.967741935483871
```

### current_date and current_timestamp

```sql
SELECT current_date FROM dual;
-- 2023-07-06

SELECT current_timestamp FROM dual;
-- 2023-07-06 10:27:01.440600 +08:00

SELECT current_timestamp(3) FROM dual;
-- 2023-07-06 10:27:14.182000 +08:00
```

### new_time

Returns the date in another time zone corresponding to a given date in a source time zone.

```sql
SELECT sysdate() bj_time, new_time(sysdate(), 'PDT', 'GMT') los_angles FROM dual;
```

### tz_offset

Returns the UTC offset for a given time zone name.

```sql
SELECT tz_offset('US/Eastern') FROM dual;
-- -04:00
```

### trunc

Truncates a date to the specified unit (year, month, day, etc.).

```sql
SELECT trunc(sysdate()) FROM dual;           -- current date at midnight
SELECT trunc(sysdate(), 'yyyy') FROM dual;   -- 2023-01-01
SELECT trunc(sysdate(), 'mm') FROM dual;     -- 2023-07-01
```

### instrb

Finds the byte position of a substring within a string.

```sql
SELECT INSTRB('CORPORATE FLOOR', 'OR') FROM DUAL;
-- 2

SELECT INSTRB('CORPORATE FLOOR', 'OR', 5, 2) FROM DUAL;
-- 14
```

### substr and substrb

`substr` extracts a substring by character position; `substrb` uses byte position.

```sql
SELECT SUBSTR('It is nice today', 5) FROM DUAL;
-- ' s nice today'

SELECT SUBSTRB('It is nice today', 5) FROM DUAL;
-- ' s nice today'

SELECT SUBSTRB('It is nice today', 5, 8) FROM DUAL;
-- ' s nice t'
```

### trim, ltrim, rtrim

```sql
SELECT trim('   aaa bbb ccc   ') FROM dual;      -- 'aaa bbb ccc'
SELECT trim('aaa bbb ccc', 'aaa') FROM dual;     -- '  bbb ccc'
SELECT ltrim('   abcdefg   ') FROM dual;         -- 'abcdefg   '
SELECT ltrim('abcdefg', 'fegab') FROM dual;      -- 'cdefg'
SELECT rtrim('   abcdefg   ') FROM dual;         -- '   abcdefg'
SELECT rtrim('abcdefg', 'fegab') FROM dual;      -- 'abcd'
```

### length and lengthb

`length` counts characters; `lengthb` counts bytes.

```sql
SELECT length(223) FROM dual;               -- 3
SELECT length('ivorysql database') FROM dual; -- 17
SELECT lengthb('ivorysql database') FROM dual; -- 17
```

### replace

```sql
SELECT replace('jack and jue', 'j', 'bl') FROM dual;
-- 'black and blue'

SELECT replace('jack and jue', 'j') FROM dual;
-- 'ack and ue'
```

### regexp_replace

Performs regular-expression-based string replacement.

```sql
SELECT regexp_replace('01234abcd56789', '[0-9]', '*#') FROM dual;
-- '*#*#*#*#*#abcd*#*#*#*#*#'

SELECT regexp_replace('01234abcd56789', '[0-9]', '*#', 2) FROM dual;
-- '0*#*#*#*#abcd*#*#*#*#*#'
```

### regexp_substr

Extracts the substring matching a regular expression.

```sql
SELECT regexp_substr('012ab34', '012', 1) FROM dual;
-- '012'

SELECT regexp_substr('012a012Ab34', '012A', 1, 1, 'i') FROM dual;
-- '012a'  (case-insensitive)
```

### regexp_instr

Returns the start position of the first (or nth) match of a regular expression.

```sql
SELECT regexp_instr('abcaBcabc', 'abc', 1);     -- 1
SELECT regexp_instr('abcaBcabc', 'abc', 1, 3);  -- 7 (third occurrence)
```

### regexp_like

Boolean-style regex match, similar to `LIKE`.

```sql
SELECT * FROM t WHERE regexp_like(value, 'abc');         -- contains 'abc'
SELECT * FROM t WHERE regexp_like(value, 'ABC', 'i');    -- case-insensitive
```

### to_number

```sql
SELECT to_number('-34,338,492', '99,999,999') FROM dual;
-- -34338492

SELECT to_number('5.01-', '9.99S');
-- -5.01
```

### to_char

```sql
SELECT to_char(sysdate()) FROM dual;
-- '2023-07-10'

SELECT to_char(sysdate(), 'mm/dd/yyyy') FROM dual;
-- '07/10/2023'
```

### to_date

```sql
SELECT to_date('20230706') FROM dual;
-- 2023-07-06

SELECT to_date('-44,0201', 'YYYY-MM-DD');
-- 0044-02-01
```

### to_timestamp and to_timestamp_tz

```sql
SELECT to_timestamp('20181102.12.34.56.025');
-- 2018-11-02 12:34:56.025000

SELECT to_timestamp_tz('2016-10-9 14:10:10.123000') FROM DUAL;
-- 2016-10-09 14:10:10.123000 +08:00
```

### to_yminterval and to_dsinterval

```sql
SELECT to_date('20110101', 'yyyymmdd') + to_yminterval('02-08') FROM dual;
-- 2013-09-01  (2 years, 8 months later)

SELECT sysdate() + to_dsinterval('0 09:30:00') FROM dual;
-- 9.5 hours from now
```

### numtodsinterval and numtoyminterval

```sql
SELECT NUMTODSINTERVAL(100.00, 'hour');
-- +000000004 04:00:00.000000000

SELECT NUMTOYMINTERVAL(1.00, 'year');
-- +000000001-00
```

### localtimestamp

```sql
SELECT localtimestamp FROM dual;
-- 2023-07-07 09:18:15.896472

SELECT localtimestamp(1) FROM dual;
-- 2023-07-07 09:18:16.100000
```

### from_tz

Converts a timestamp to a timestamp-with-time-zone.

```sql
SELECT FROM_TZ(TIMESTAMP '2000-03-28 08:00:00', '3:00') FROM DUAL;
-- 2000-03-28 13:00:00.000000 +08:00
```

### sys_extract_utc

Converts a `TIMESTAMP WITH TIME ZONE` value to UTC.

```sql
SELECT sys_extract_utc(TIMESTAMP '2000-03-28 11:30:00.00 -8:00') FROM dual;
-- 2000-03-28 19:30:00.000000
```

### sessiontimezone

```sql
SELECT sessiontimezone() FROM dual;
-- Asia/Shanghai

SET timezone = 'Asia/Hong_Kong';
SELECT sessiontimezone() FROM dual;
-- Asia/Hong_Kong
```

### hextoraw

```sql
SELECT hextoraw('abcdef') FROM dual;
-- \xabcdef
```

### uid

Returns the session's user OID.

```sql
SELECT uid() FROM dual;
-- 10
```

### USERENV

Returns information about the current user environment.

```sql
SELECT userenv('isdba') FROM dual;
-- TRUE

SELECT userenv('sessionid') FROM dual;
-- 1
```

### asciistr

Returns the ASCII representation of a string, escaping non-ASCII characters.

```sql
SELECT asciistr('Hello, World!') FROM dual;
-- 'Hello, World!'

SELECT asciistr('你好') FROM dual;
-- '\4F60\597D'
```

### to_multi_byte and to_single_byte

```sql
SELECT to_multi_byte('1.2'::text);
-- '１．２'  (half-width → full-width)

SELECT to_single_byte('１．２');
-- '1.2'   (full-width → half-width)
```

### compose and decompose

```sql
SELECT compose('a' || chr(768)) FROM dual;
-- 'à'  (base character + combining mark → composite)

SELECT asciistr(decompose('é')) FROM dual;
-- 'e\0301'  (composite → base character + combining mark)
```
