# Developers

This section is for developers who want to understand IvorySQL internals, contribute to the project, or explore experimental features under active development.

IvorySQL is built on top of PostgreSQL and extends it with Oracle compatibility. The developer resources here cover the architecture of those extensions, how to contribute code and documentation, and experimental capabilities that are available for early testing.
